If you're already familiar with <a href="http://www.msgplus.net/" title="MsgPlus Homepage">MsgPlus</a> - a feature packed MSN Messenger addin - you may or may not be aware of the fact that MsgPlus can itself install add-ons of its own.

One such add-on is <a href="http://www.stuffplug.com" title="StuffPlug NG Homepage">StuffPlus NG</a>, adding a rather smaller array of features but two which I already find indispensable:
<dl>
<dt>Timestamps in conversations</dt>
<dd>No more losing track of messages by displaying the time of arrival/dispatch next to each message</dd>
<dt>Tabbed Messages</dt>
<dd>For the tab-fiend in all of us (see: <a href="http://www.getfirefox.com" title="Mozilla Firefox">Firefox</a>, <a href="http://www.opera.com" title="Opera Web Browser/Suite">Opera</a> converts). Rather than have a gazillion chat windows open at once, have them all in one and separated by tabs. A little buggy, but rather nice all the same</dd>
</dl>

Verrry nice.
